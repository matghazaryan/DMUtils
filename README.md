# DMUtils
The most used utils file for fastest Android development

Add it in your root build.gradle at the end of repositories:

    allprojects {
        repositories {
          ...
          maven { url 'https://jitpack.io' }
        }
      }
      
Add the dependency

	dependencies {
		implementation 'com.github.pmbfish40:DMUtils:1.0.2'
	}
